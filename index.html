Projects

Maintenance of the Halo España Website

December 2023 - Present

I am responsible for the installation, configuration, and maintenance of the web server for the Halo España community. The website is based on WordPress and runs on a LAMP infrastructure. My role focuses on technical management, ensuring everything operates smoothly, while also contributing articles to the website in my spare time.

The Flood: The Forgotten Exile

April 2023 - Present

I worked on adapting the original version of the mod "The Flood: The Forgotten Exile" from Halo: Custom Edition to Halo: The Master Chief Collection. My responsibilities included adapting and recompiling the map, fixing bugs, and programming the code for the cooperative version of the mod, as Halo CE did not originally support this feature. I also maintain the mod on the Steam Workshop, handle feedback, and ensure its proper functionality.

3D Printing Remote Server

May 2022

A Linux server based on the Raspbian distribution, combined with Octoprint and automated using Shell scripting. The setup allows connections either via DDNS over the Internet or through an OpenVPN server. It also includes an FTP server and a video streaming system for real-time print monitoring. Hardware used includes a Raspberry Pi B2 and a Creality Ender 3 Pro printer. This project earned the highest grade in my vocational training final project.
